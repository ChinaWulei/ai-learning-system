# 双人足球对战

一个独立的浏览器足球小游戏项目，不依赖原学习系统。

## 运行

直接用浏览器打开 `index.html` 即可开始练习模式。

如果要测试复制代码、WebRTC 等浏览器能力，建议在本目录启动本地静态服务：

```bash
python -m http.server 8088
```

然后访问：

```text
http://127.0.0.1:8088
```

## 玩法

- 移动：`WASD` 或方向键
- 射门：空格
- 练习模式：蓝队由你控制，红队由电脑控制
- 双标签模式：同一台电脑打开两个标签页，一个创建房间，一个输入房间码加入
- 联机模式：主机生成主机码，客队粘贴主机码生成客队码，再由主机确认客队码
